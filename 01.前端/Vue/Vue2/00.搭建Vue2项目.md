# 搭建[Vue2](https://v2.cn.vuejs.org/v2/guide/)项目

## 1、全局安装[vue-cli](https://cli.vuejs.org/zh/)

```bash
npm install -g @vue/cli 
```

看 `vue-cli` 版本：`vue -V` 或 `vue --version`

## 2、使用vue-cli初始化一个项目

```bash
vue create project-name
```

> project-name：项目名称。不能有大写字母，特殊字符。且不能和项目要安装的依赖重名。
